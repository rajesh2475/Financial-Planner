# ITR Navigator – Tax Planner & Comparison

ITR Navigator is a **Flask-based financial planner and tax comparison tool** designed to help employees and individuals make smarter tax decisions.  
It compares **Old vs New Regimes**, highlights deductions, and projects savings to guide users through **Income Tax Return (ITR) planning** with clarity and confidence.

---

## 🚀 Features
- 🧾 **Tax Planner Form**: Enter salary, HRA, 80C, NPS, 80D, Home Loan, and other deductions.
- 📊 **Regime Comparison**: Side-by-side view of Old vs New Regimes with taxable income, deductions, base tax, and total tax (incl. cess).
- 💡 **Suggestions**: Highlights underutilized deductions and potential savings.
- 🔍 **Tooltips**: Info buttons explain how each number is calculated.
- 🏠 **Dashboard Layout**: Left navigation with Home, Tax Planner, Reports, and Pricing.
- ✅ **ITR Focus**: Helps employees prepare inputs for ITR filing with confidence.

---

## 🛠️ Tech Stack
- **Backend**: Python 3, Flask
- **Frontend**: Jinja2 templates, Bootstrap 4
- **Deployment**: Works locally (`flask run`) and can be hosted on free platforms like Render, Railway, or PythonAnywhere.
